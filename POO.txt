TIPOS DE DADOS EM JAVA
JAVA tem varios tipos de dados, e tem tipos de dados primitivos que todo o desenvolvedor precisa saber

byte = tipo de dado que recebe somente byte. Menor byte que pode receber
é -128 e o maior é 127

Short menor -32768 e maior é 32767

Integer é pra trabalhar com tipos inteiros. Maior é 2147483647 e o menor -2147483648

Long, valores MUITO grandes e ñ vou escreverr aqui.É basicamente um inteiro (integer) gigante. Long 92337203684775807. Precisa colocar um L no final

Float pra valores flutuantes  precisa colocar o F no final

double é um float mais preciso

Temos o String, que é comum PRA TODOS. Armazena texto.
char: Também é texto, mas textos pequenos. Usados pra gguardar carácter. Não colocamos entre aspas duplas
